# Project name: Corporate Employee Registry portal

## Overview
A Professional-grade **Corporate Employee Registry Portal** designed to streamline workforce data management. This application provides a centralized interface for HR and administrative teams to capture employee information and manage associated corporate assets through an integrated Active Database Registry Matrix.

## Key features
**Employee data Capture** Captures essential staff profiles including Full Name, Email, and Departmental affiliation.
**Integrated Assets Management** Features a specialized "Add Asset" workflow, allowing administrators to link corporate equipment and resources directly to specific employee profiles.
**Active Database Registry Matrix** Maintains a real-time, structured overview of all registered employees and their corresponding registry data, ensuring data integrity and quick reference.
**Streamlined UI** A clean, efficient interface optimized for high- volume data entry and administrative oversight.

## Technologies Used
**Backend**: PHP
**Frontend**: HTML5, CSS3
**Database**: SQL

## How to Run
1. Ensure your local server environment(XAMPP) is active.
2. Clone this repository into your server's root directory (htdocs).
3. Import the provided database schema to initialize the Registry Matrix.
4. Acces the portal via 'http://localhost/[your-folder-name]/index.php'.

## Project Structure
* 'index.php': The primary portal interface for registration and asset assignment.
* 'db_connect.php': Configuration settings for the Active Database registry Matrix.
